# Human-Centric AI Decision Support System

ระบบปัญญาประดิษฐ์ที่ปรึกษาเชิงกลยุทธ์และอารมณ์ ออกแบบมาเพื่อวิเคราะห์ปัญหาซับซ้อนโดยคำนึงถึงมิติความเป็นมนุษย์

## โครงสร้างโปรเจค

```
/workspace
├── core/                      # แกนหลักของระบบ
│   ├── cognitive_engine/      # เครื่องยนต์การคิดและประมวลผล
│   │   ├── engines/           # ระบบประมวลผลหลัก
│   │   └── humanity_response.py
│   ├── symbolic_logic_system.py    # ระบบตรรกะสัญลักษณ์
│   └── multiverse_reasoning_system.py  # ระบบเหตุผลหลายมิติ
│
├── modules/                   # โมดูลเสริม
│   ├── analysis_planner_pack.py       # ระบบวางแผนการวิเคราะห์
│   ├── advanced_analysis_planner_pack.py  # ระบบวางแผนขั้นสูง
│   ├── personality_analyzer.py        # ระบบวิเคราะห์บุคลิกภาพ
│   └── humanity_dynamic_response.py   # ระบบตอบสนองแบบไดนามิก
│
├── utils/                     # เครื่องมือช่วยเหลือ
│   └── humanity_tools.py      # เครื่องมือจัดการมิติมนุษย์
│
├── data/                      # ข้อมูลและฐานความรู้
└── README.md                  # เอกสารนี้
```

## ความสามารถหลัก

1. **การวิเคราะห์ปัญหาซับซ้อน** - แยกองค์ประกอบปัญหาและประเมินทางเลือก
2. **การตัดสินใจเชิงจริยธรรม** - คำนึงถึงผลกระทบทางอารมณ์และสังคม
3. **การจำลองสถานการณ์** - ทดสอบผลลัพธ์ในสถานการณ์ต่างๆ
4. **การปรับรูปแบบคำตอบ** - สร้าง response ได้หลายรูปแบบตามผู้รับ

## การใช้งานเบื้องต้น

```python
from core.cognitive_engine.engines.core import CognitiveEngine
from modules.analysis_planner_pack import StrategicPlanner

# เริ่มต้นเครื่องยนต์
engine = CognitiveEngine()

# วางแผนการวิเคราะห์
planner = StrategicPlanner()
# ใช้งาน planner ตามความต้องการ
```

## การขยายความสามารถ

สามารถเพิ่มโมดูลใหม่ได้โดย:
1. สร้างไฟล์ในโฟลเดอร์ `modules/` หรือ `utils/`
2. นำเข้าและใช้งานใน `core/`
3. อัปเดตเอกสารการใช้งาน

## หมายเหตุ

ระบบนี้ใช้ LLM เป็นเครื่องมือภายใต้โครงสร้างการคิดที่ควบคุมได้ เพื่อให้ได้คำตอบที่มีเหตุผล ปลอดภัย และเข้าใจมิติความเป็นมนุษย์
